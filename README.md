# Tugas 1 - Mobile Development (GDGOC Telkom)

Repository ini berisi tugas "Coding Bebas" menggunakan bahasa pemrograman Dart.

## Fitur Code
- ✅ Penggunaan Tipe Data (`var`, `final`, `const`)
- ✅ Logika Percabangan (`if`, `else if`)
- ✅ Perulangan (`for` loop)
- ✅ Fungsi dengan `return` dan `void`

## Cara Menjalankan
Pastikan Dart SDK sudah terinstall, lalu jalankan perintah:
dart day1.dart
